How to Run the Survey Webpage:
The steps for running the webpage assume the repository code is cloned to the local computer. Clone this repository if not yet done.
The steps also assume that node js has been installed on the local computer. Installation can be found here: https://nodejs.org/en/

1. Open the lab folder.
2. Open a terminal from the lab folder.
3. Enter "node data/server.js" to run the server.
4. A message should show up saying Server running at http://localhost:3000, copy and paste http://localhost:3000 into a search tab.
5. The survey webpage should now be running,
